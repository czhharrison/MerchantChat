```python
# 修复前
def estimate_ctr(title: str, keywords: List[str]) -> Dict[str, Any]:

def analyze_competitor_title(competitor_title: str, our_keywords: List[str]) -> Dict[str, Any]:

# 修复后
def estimate_ctr(title: str, keywords: List[str] = None) -> Dict[str, Any]:
    if keywords is None or len(keywords) == 0:
        keywords = [k for k in jieba.cut(title) if len(k) > 1][:5]

def analyze_competitor_title(competitor_title: str, our_keywords: List[str] = None) -> Dict[str, Any]:
    if our_keywords is None or len(our_keywords) == 0:
        our_keywords = competitor_keywords[:3]
```

```python
# 在单独线程中延迟打开浏览器（仅一次）
def open_browser_once():
    time.sleep(3)  # 等待streamlit启动
    try:
        webbrowser.open('http://localhost:8501')
        print("✓ 浏览器已自动打开")
    except Exception as e:
        print(f"打开浏览器失败: {e}")

import threading
browser_thread = threading.Thread(target=open_browser_once)
browser_thread.daemon = True
browser_thread.start()

# 启动streamlit（禁用自动打开浏览器，避免重复）
subprocess.run([
    sys.executable, '-m', 'streamlit', 'run', 'streamlit_app.py',
    '--server.headless', 'true',  # 禁用streamlit自动打开
    '--browser.gatherUsageStats', 'false'  # 禁用统计数据收集
])
```

**根本原因：**
- 单项工具测试的结果没有保存到session state中
- 页面重新渲染时结果丢失
- 缺乏结果持久化机制

**修复方案：**
- **session state扩展**: 在`init_session_state()`中添加`tool_results`字典存储
- **结果保存**: 每个工具生成结果时自动保存到对应的session state
- **结果显示**: 页面重新访问时自动显示保存的结果
- **清除功能**: 为每个工具添加独立清除按钮和全局清空功能

**技术实现：**
```python
# session state初始化

# 结果保存示例（以竞品分析为例）
if st.button("分析竞品") and competitor_input and our_keywords_input:

# 结果显示逻辑

    # 显示完整的分析结果，包括LLM深度分析
    # 显示输入参数信息
```
